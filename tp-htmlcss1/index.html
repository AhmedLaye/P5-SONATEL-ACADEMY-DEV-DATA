Connaissances générale

  • Internet

    • L'Internet est un réseau mondial d'ordinateurs interconnectés qui permettent la communication et l'échange d'informations à travers le monde. Il s'agit d'un réseau décentralisé qui permet la transmission de données à travers différents réseaux et dispositifs.

  • Web

    • Le Web, ou World Wide Web (WWW), est une application spécifique de l'Internet qui permet l'accès à des pages web à travers des navigateurs web. Le Web est composé de pages web interconnectées, qui peuvent contenir du texte, des images, des vidéos, des liens hypertextes, et d'autres types de contenu.

  • Protocole

    • Un protocole est un ensemble de règles et de normes qui régissent la communication entre des dispositifs informatiques. Les protocoles sont utilisés pour assurer que les dispositifs sont capables de communiquer entre eux de manière cohérente et standardisée.

  • W3S

    • Le W3C (World Wide Web Consortium) est un organisme international qui établit des normes et des protocoles pour le développement du Web. Le W3C a pour mission de développer des technologies et des standards ouverts pour le Web, afin de promouvoir l'interopérabilité, l'accessibilité, et l'innovation.

Fonctionnement d'un site web

  • Les outils necesaire pour faire focntionner un site Web

    • Un serveur web : c'est un logiciel qui permet de stocker et de distribuer les fichiers et les pages web qui composent un site. Un nom de domaine : c'est l'adresse web qui permet aux utilisateurs d'accéder au site, par exemple www.monsite.com. Un système de gestion de contenu (CMS) : c'est un logiciel qui permet de gérer et de publier facilement du contenu sur un site web, sans avoir besoin de connaissances techniques approfondies en programmation. Un langage de programmation : il existe plusieurs langages de programmation qui sont utilisés pour développer des sites web, comme HTML, CSS, JavaScript, PHP, Python, Ruby, etc. Des outils de développement : il s'agit de logiciels qui permettent de développer et de tester le site web, comme un éditeur de code, un environnement de développement intégré (IDE), un navigateur web, etc..

  • Les langages Nécessaire pour faire fonctionner un site Web

    • Les langages nécessaires pour faire fonctionner un site web dépendent du type de site web que l'on souhaite créer et des fonctionnalités que l'on souhaite y intégrer. Voici quelques-uns des langages les plus couramment utilisés :

    HTML : c'est un langage de balisage qui permet de structurer le contenu d'une page web. CSS : c'est un langage qui permet de mettre en forme et de styliser une page web. JavaScript : c'est un langage de programmation qui permet d'ajouter des fonctionnalités interactives à une page web, comme des animations, des effets visuels, des formulaires, etc. PHP : c'est un langage de programmation côté serveur qui permet de générer du contenu dynamique, comme des pages web personnalisées en fonction des données saisies par l'utilisateur. SQL : c'est un langage de requête qui permet de stocker, de récupérer et de manipuler des données dans une base de données, comme des informations d'utilisateur ou des produits d'une boutique en ligne.

Balises et attributs

  • Balise Inline vs Block

    • Une balise inline est une balise qui est utilisée pour définir des éléments en ligne, comme un texte ou une image, qui sont insérés à l'intérieur d'un bloc de texte. Les balises inline n'affectent pas la structure de la page et ne provoquent pas de saut de ligne. Exemples : <span> <a>
    Une balise block est une balise qui est utilisée pour définir des éléments qui sont affichés sous forme de blocs distincts, comme des paragraphes, des titres, des listes, etc. Les balises block occupent toute la largeur disponible et provoquent un saut de ligne. Exemples : <div>, <p>, <h1> h6...

  • Attribut pour une Balise

    • Un attribut est une caractéristique supplémentaire que l'on peut ajouter à une balise pour spécifier des informations telles que la couleur, la taille, le lien, l'image, etc. Les attributs sont définis à l'intérieur de la balise d'ouverture, entre parenthèses et sont généralement sous la forme "nom=valeur". Exemple : <a href="https://www.example.com">Lien</a>, où "href" est l'attribut qui spécifie l'URL du lien.

Structure d'une page HTML

Doctype

La première ligne d'un document HTML doit contenir la déclaration Doctype, qui indique au navigateur comment interpréter le code HTML.

Texte et paragraphe

Le texte est généralement inséré dans des balises de paragraphe (<p>). Il est possible de mettre en forme le texte en utilisant des balises comme <em> pour l'italique ou <strong> pour le gras.

Titre

Le titre d'une page HTML est spécifié avec la balise <title>, qui doit être placée dans la section <head> du document. Le titre s'affiche dans l'onglet du navigateur et dans les résultats de recherche.

Liste

Il existe deux types de listes en HTML : les listes ordonnées (<ol>) et les listes non ordonnées (<ul>). Les éléments de la liste sont spécifiés avec la balise <li>.

Liens

Les liens sont spécifiés avec la balise <a>. Il est possible de créer des liens externes avec l'attribut href, des liens internes avec l'attribut href et le chemin relatif ou absolu, et des ancres avec l'attribut id.

Lien Externe

Pour créer un lien externe, il suffit de spécifier l'URL du site cible dans l'attribut href. Par exemple :

<a href="https://www.example.com">Lien vers Example</a>

Liens (suite)

Lien Interne

Pour créer un lien interne, il faut spécifier l'emplacement de la page cible en utilisant un chemin relatif ou absolu. Par exemple, si la page cible se trouve dans le même dossier que la page courante :

<a href="page2.html">Lien vers la page 2</a>

Si la page cible se trouve dans un dossier différent :

<a href="dossier/page3.html">Lien vers la page 3</a>

Ancre

Une ancre est un lien qui permet de se déplacer vers une section spécifique d'une même page. Pour créer une ancre, il faut utiliser l'attribut id sur la balise correspondante :

<h3 id="lien">Lien</h3>

Le lien vers cette ancre se fera en utilisant le symbole # suivi de l'identifiant :

<a href="#lien">Lien vers la section sur les liens</a>

Images

Les images sont insérées avec la balise <img>. Il est possible de spécifier l'emplacement de l'image en utilisant un chemin relatif ou absolu, ainsi que les dimensions et le texte alternatif avec les attributs src, width, height et alt.

Insertion

Voici un exemple d'insertion d'une image :

<img src="images/image1.jpg" alt="Image 1" width="500" height="300">

Cliquable

Pour rendre une image cliquable, il suffit de l'entourer d'une balise <a> :

<a href="page2.html"><img src="images/image2.jpg" alt="Image 2" width="300" height="200"></a>
